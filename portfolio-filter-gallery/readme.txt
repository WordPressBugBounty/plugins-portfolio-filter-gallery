Create filterable portfolio galleries. Supports photo, and video with category filters, masonry layouts, Gutenberg block and Elementor widget.

== Description ==

**Portfolio Filter Gallery** is a powerful and easy-to-use WordPress gallery plugin designed for photographers, designers, agencies, and anyone who needs to display their work in an organized, professional way. Whether you are building a photography portfolio, a design showcase, or a simple image gallery, this plugin gives you the tools to present your content beautifully.

The plugin lets you organize images and videos into filter categories so visitors can browse your work by topic, project type, or any grouping you choose. Instead of scrolling through dozens of images, your visitors simply click a filter button to narrow down what they see. It is a practical way to keep large galleries manageable and user-friendly.

= Works with Gutenberg and Elementor =

Starting from version 2.2.0, Portfolio Filter Gallery includes a dedicated Gutenberg block and a native Elementor widget. You no longer need to copy and paste shortcodes when working with these editors.

**Gutenberg Block Editor** - Search for "Portfolio Filter Gallery" in the block inserter, pick your gallery from the dropdown, and see a live preview right inside the editor. The block renders your actual gallery with all its filters, layout, and styling so you know exactly what the published page will look like.

**Elementor Page Builder** - Drag the "Portfolio Filter Gallery" widget from the Elementor panel into your layout. Select a gallery, toggle options, and watch the gallery appear in the Elementor canvas in real time. No shortcode pasting, no guesswork.

Both integrations include optional overrides for columns, hover effects, and filter visibility, giving you per-page control without changing your original gallery settings.

For page builders like WPBakery, Divi, and Beaver Builder, you can still use the standard shortcode in any text or HTML block.

= Why Portfolio Filter Gallery =

Building a gallery in WordPress should not be complicated. This plugin keeps the setup straightforward: create your filter categories, upload your images, assign filters, and publish. The admin interface uses drag-and-drop ordering, bulk selection, and a clean layout that stays manageable even with hundreds of images.

On the frontend, galleries load with smooth CSS3 animations and adapt to any screen size. The masonry layout fits images of different dimensions together naturally, while the grid layout gives you a uniform, structured look. Both layouts work well on desktop, tablet, and mobile.

= How to Set Up Your Gallery =

1. Go to **Portfolio Gallery** in your WordPress admin menu and click **Filters**
2. Add the filter categories you need, such as "Web Design", "Photography", or "Branding"
3. Click **Add New** under Portfolio Gallery to create a new gallery
4. Upload your images, enter titles and descriptions, and assign filters to each item
5. Configure your layout, column count, hover effects, and colors in the gallery settings
6. Publish the gallery and add it to any page using the Gutenberg block, Elementor widget, or shortcode

The shortcode format is:
`[Portfolio_Gallery id="123"]`

Replace 123 with the ID shown in your gallery editor.

= Photo Gallery Masonry Layout =

The masonry layout arranges images in a column-based grid where each image retains its original aspect ratio. Tall and wide images sit next to each other without awkward gaps or forced cropping. This layout works especially well for photography portfolios and design showcases where images come in varying dimensions.

The masonry grid recalculates on window resize, so it looks good on any screen. Filter transitions are animated with CSS3, keeping the experience smooth and responsive.

= Custom Links on Gallery Items =

Every gallery item can have a custom link. This is useful when you want to direct visitors to a project detail page, a client website, or a product listing. You can set each link to open in the same tab or a new tab depending on how you want your navigation to work.

== Installation ==

**Install from WordPress Dashboard**

1. Go to your WordPress Dashboard, then Plugins, then Add New
2. Search for "Portfolio Filter Gallery"
3. Click "Install Now" and then "Activate"

**Manual Installation**

1. Download the plugin ZIP file from WordPress.org
2. Upload the `portfolio-filter-gallery` folder to `/wp-content/plugins/`
3. Go to the Plugins page in your WordPress dashboard and activate it

**After Activation**

1. Navigate to Portfolio Gallery and click Filters to create your filter categories
2. Click Add New under Portfolio Gallery to create your first gallery
3. Upload images, fill in titles and descriptions, and assign filters
4. Set your preferred layout, column count, and colors
5. Add the gallery to any page using the Gutenberg block, Elementor widget, or shortcode

== Frequently Asked Questions ==

= Is this gallery plugin free to use? =

Yes. The free version includes everything you need to create filterable image galleries with masonry and grid layouts, video embedding, and page builder integration. A [premium version](https://awplife.com/wordpress-plugins/portfolio-filter-gallery-wordpress-plugin/) is available for users who need advanced features like multi-select filtering, search, watermarks, and additional hover effects.

= How do I add the gallery to a page? =

You have three options. In the Gutenberg block editor, search for "Portfolio Filter Gallery" in the block inserter and select your gallery. In Elementor, drag the Portfolio Filter Gallery widget into your layout. For other editors, copy the shortcode from your gallery and paste it into any text block.

= How do I create and manage filters? =

Go to Portfolio Gallery then Filters in your WordPress dashboard. This page lets you add, edit, reorder, and delete filter categories independently from your galleries. Once created, you can assign any filter to any image across all your galleries.

= Can I add YouTube and Vimeo videos to the gallery? =

Yes. When adding a gallery item, choose the video option and enter the YouTube or Vimeo URL. Videos appear as thumbnails in the gallery and open via external links when clicked. You can see a working example on the [video gallery demo page](https://awplife.com/demo/portfolio-filter-gallery-premium/portfolio-video-gallery/).

= Does the gallery work on mobile devices? =

Yes. All gallery layouts are fully responsive and adapt to phones, tablets, and desktops. The masonry and grid layouts reflow automatically based on screen width, and filter buttons remain accessible on touch screens.

= Is there a limit to how many images I can add? =

No. You can add as many images and videos as your hosting allows. The plugin is optimized for large galleries with chunked saving to handle hundreds of items reliably.

= Can I use multiple galleries on the same page? =

Yes. Each gallery generates its own shortcode and operates independently. Filters, settings, and layouts are unique to each gallery instance.

= Can I reorder images after uploading? =

Yes. The gallery editor supports drag-and-drop reordering. You can also use Shift+Click to select multiple images and move them together.

= Does it work with the Gutenberg block editor? =

Yes. Version 2.2.0 introduced a native Gutenberg block. You can search for it in the block inserter, select your gallery, and see a live server-side preview directly in the editor.

= Does it work with Elementor? =

Yes. Version 2.2.0 added a native Elementor widget. You can drag it from the widget panel, choose a gallery, and the preview renders live inside the Elementor builder canvas.

= Does it work with other page builders? =

Yes. For WPBakery, Divi, Beaver Builder, and other page builders, paste the gallery shortcode into any text, HTML, or shortcode block.

= Is the plugin GDPR compliant? =

Yes. Portfolio Filter Gallery does not collect any user data, does not use cookies, and does not make external requests. All plugin assets load from your own WordPress installation.

= Does it support translation? =

Yes. The plugin is fully translation-ready with a standard text domain. It works with WPML, Polylang, Loco Translate, and other translation tools.

= Where can I get help if something is not working? =

Visit the [WordPress Support Forum](https://wordpress.org/support/plugin/portfolio-filter-gallery/) for community help. The development team monitors the forum and typically responds within one business day. Premium users can access priority support through the [developer website](https://awplife.com/wordpress-plugins/portfolio-filter-gallery-wordpress-plugin/).
